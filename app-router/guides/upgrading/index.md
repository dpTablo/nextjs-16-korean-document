# 업그레이드 가이드

**문서 버전:** 16.1.2
**최종 업데이트:** 2025-04-15

버전별 가이드를 따라 최신 Next.js 버전으로 업그레이드하는 방법을 알아보세요:

- [Codemods](./codemods.md)
  - 새로운 기능이 출시될 때 코드모드를 사용하여 Next.js 코드베이스를 업그레이드합니다.
- [버전 14](./version-14.md)
  - Next.js 애플리케이션을 버전 13에서 14로 업그레이드합니다.
- [버전 15](./version-15.md)
  - Next.js 애플리케이션을 버전 14에서 15로 업그레이드합니다.
- [버전 16](./version-16.md)
  - Next.js 애플리케이션을 버전 15에서 16으로 업그레이드합니다.
