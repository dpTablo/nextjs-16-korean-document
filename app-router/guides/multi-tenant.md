# Next.js에서 멀티테넌트 앱을 구축하는 방법

**문서 버전:** 16.1.2
**최종 업데이트:** 2025-04-15

---

## 내용

여러 테넌트를 서비스하는 단일 Next.js 애플리케이션을 구축하려는 경우, 권장 아키텍처를 보여주는 [예제를 구축했습니다](https://vercel.com/templates/next.js/platforms-starter-kit).

이 예제는 다음을 포함합니다:
- 커스텀 도메인 지원
- 테넌트별 설정 및 데이터 분리
- 동적 라우팅을 통한 멀티테넌시 구현
- Vercel Platforms를 활용한 배포 전략

자세한 구현 방법은 [Platforms Starter Kit](https://vercel.com/templates/next.js/platforms-starter-kit)를 참조하세요.
